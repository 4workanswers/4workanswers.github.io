                               Application Overview and Screen Shots

            The application has 3 main functions:





  1. Send appointment confirmation and pick-up reminder text messages

    • Use tailoring appointment entries(labeled: Name, Cell#, Date, Time) to send an appointment confimation text message to the customer.

    • Use pick-up reminder entries(labeled: Name, Cell#, Date Ready, Ticket#(s)) to send a text message to the customer reminding them that their order has been complete since the Ready Date.

    • Reduce the occurance of appointment no-shows and provide a courtesy to the customer.

  2. Manage tailoring appointments made and pick-up reminders sent

    • Records tailoring appointment and pick-up reminder data in text files and gives each appointment/reminder an indiviual ID number.

    • Parse appointment or reminder data with a click of the View All Appointments or View All Reminders button, respectively.

    • Remove a reminder or appointment by entering its unique ID number in the entry field labeled Appt# or Rem# and clicking the Delete Appointment or Delete Reminder button.

  3. Help manage store's garment-rack space

    • Utilization of the Pick-Up Reminder function increases the likelihood
      of timely order pick-up thereby preventing completed-order inventory
      from growing and limiting garment-rack space.                                                                                                                                                                                                                             Note: About page includes screen shots of the app in action and info about its functions

    • While paper reciepts and reminders can easily be lost, misplaced or
      damaged, text messages persist on the customer's cell phone and must
      be actively viewed before being deleted or lost.


      As mentioned earlier, this application is capable of sending
2 distinct text messsages: one with tailoring/alterations
appointment information and another with a dry-cleaning
pick-up reminder. The messages as they will appear to the
recipiant are pictured below; the appointment reminder is on
the left and the pick-up reminder is on the right.
I've redacted the last 4 digits of the phone number
for the purpose of privacy; eventhough the number
can be found in a simple search engine search,
I just thought it wouldn't be appropriate to use it in
the marketing of our application.

      The appointment reminder text message includes the date
and time information entered by the user in the Date and Time
fields of the application's user-interface. The pick-up
reminder text message includes ready-date information
entered by the user in the Date Ready field of the user-interface.
Both messages include "DO NOT REPLY" because replies
to the text messages are not monitored. For this same reason,
the store telephone number is given in case customers have
questions or concerns. The store number displayed in
the text message images above is for a particular Cleaner
(Gentle Care Cleaners) that is currently using our software
and can easily be personalized, along with any
other part of the message, for other Cleaners.
